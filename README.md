# **AI Fake News Detection System**

## *Project Description*
AI Fake News Detection System is a machine learning-based web application that identifies whether a news article is real or fake. The system analyzes the text using Natural Language Processing (NLP) techniques and a trained machine learning model to predict the authenticity of news content. The application uses a Flask backend for prediction and a web-based frontend for user interaction.

## *Technologies Used*

Python, Flask, Scikit-learn, Pandas & NumPy, HTML, CSS, JavaScript, TF-IDF Vectorization

## *Features*

◦ Detects fake and real news articles

◦ Machine learning-based prediction

◦ Fast API prediction using Flask

◦ Simple and interactive web interface

◦ Probability-based result output
